# LinkedIn-Clone
This is a basic clone implementation of the Linked application.

### USER INTERACTION

1. A user can sign up and subsequently sign in using thein credentials.
2. A user is able to post on their feed.
3. A user can logout.


### TECHNOLOGIES USED

_FRAMEWORKS_

* React

* Redux

* _JavaScript_

* _ES6_
  
* Firebase


_FRAMEWORK_

* _Material UI_
